## Project Overview
This project analyzes customer sentiment from Flipkart product reviews of the iPhone 15 (128GB).
The goal is to understand customer opinions using sentiment analysis techniques.

## Tools & Libraries Used
- Python
- Requests
- BeautifulSoup
- Pandas
- TextBlob
- Matplotlib

## Steps Performed
1. Web scraping customer reviews from Flipkart
2. Data cleaning and preprocessing
3. Sentiment analysis using TextBlob
4. Data analysis and visualization
5. Insights and recommendations

## Key Insights
- Majority of reviews are positive
- Positive reviews have higher ratings
- Negative reviews are fewer but longer in length
- Review length helps identify dissatisfaction

## Dataset
- ~289 customer reviews collected

## Conclusion
Sentiment analysis provides deeper insights than ratings alone and helps in understanding customer behavior.
